プラグイン管理

bundleやpathgenなど色々迷ったが、最終的にvim-plugにした

.vimrcファイルに欲しいプラグインの"/<リポジトリ名>"を入力して

:PlugInstall install plugin added in .vimrc.(実行前に:source $MYVIMRCを実行する）

:PlugUpdate Updata Plugin

:PlugClean Clean Plugin that was removed from .vimrc.

プラグイン情報

plugin description
mru.vim

View recently opened file history.

/mで作動するように設定している。
vim-commentary

help add comments.

コメントアウトしたい箇所を選択してgccでOK.外したいときも同様でOK.
vim-easymotion

move cursol at will.

<ctrl f>で作動するように設定している。
vim-ack

ackをvimから使用できるようにした。

":Ack --python Animal" => pythonファイルからAnimal文字列を含む物を列挙

YouCompleteMe 候補が出てくるのでtabまたはctrl+n,ctrl+pで選択を行う。
tagbar

外部機能のUniversalCtagを使うのでインストールした。

<Leader>tで関数一覧表示。<Leader>Tで現在の関数名表示するようにした。

あとファイル更新時に自動でタグを更新するようにした。拡張子は必要となったら随時追加する必要あり

fern

コマンド 動作
e ファイルを開く
E ファイル垂直分割でを開く
l フォルダを展開
h フォルダを閉じる
Ctrl+m 選択フォルダに潜る
Ctrl+h 親ディレクトリに移動
N ファイルを新規作成
K ディレクトリを新規作成
D ファイルを削除
R ファイルをリネーム
m ファイルを移動
y ファイルパスをコピー
! 隠しファイルを表示/非表示

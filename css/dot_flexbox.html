display:flex

親要素にdisplay:flexを指定すると子要素のボックスが横並びになる。

子要素にflex:1;と指定するとその子要素が親要素の幅に合わせて柔軟に広がる。

親要素の事をflex container,子要素の事をflex itemと呼ぶ

flex containerの軸について

コンテナの左上を起点として右方向に主軸、下方向に交差軸が設定される。

flex-direction

主軸の方向を決めるプロパティでデフォルト値はrowである。row-reverseとすると主軸の方向が逆になる。

flex-direction:columnとすると主軸と交差軸が入れ替わる。なのでflex itemが上から下に順番に並べられる。

flex-direction:column-reverseとするとitemが下から上に並べられる。

itemを主軸に沿ってどう揃えるか指定。justify-contentプロパティ

flex-startとすると主軸の始点（左）に揃えて並べる。

flex-endとすると主軸の終点（右）に揃えて並べる。

centerとすると主軸の中央に揃えて並べる。

space-betweenアイテムの間を均等にして並べる。

itemを交差軸に沿ってどう揃えるか指定。align-itemsプロパティ

flex-startとすると主軸の始点（左）に揃えて並べる。

はみ出した分をどうするか指定flex-wrapプロパティ

デフォルトだとnowrapに設定されており、これは主軸に収まるようにitemを縮める事。

wrapとするとはみ出した分の要素を交差軸に沿って並べてくれる。

wrap-reverseとしたら、交差軸の反対方向に並べてくれる。

itemを交差軸方向に沿ってどう揃える指定align-contentプロパティ

値はjustify-contentと同じ

flex-directionとflex-wrapを一括指定する。flex-flow

flex-flow: row wrapと一括指定できる。

flex itemに付けられる。プロパティ

交差軸に対する配置を決める。align-selfプロパティ

flex-start交差軸の始点、flex-endで終点,centerで中央

flex itemの順番を決めるプロパティorderプロパティ

初期値が0である。あるアイテムをorder:1などとするとそのアイテムが最後に並べられる。

order: -1;とするとそのアイテムが先頭に並べられる。

flexコンテナ内でのflex itemの幅（高さ）を指定する。flex-basisプロパティ

デフォルトだとautoでこれはその要素の大きさを決めるルールに従う。（height,widthとかコンテンツとか）

主軸方向に余白が有る際にどれぐらい伸ばすかflex-growプロパティ

flex-grow:1;flex-grow:2;でそれぞれの要素を決めたら、余白を1:2の割合で埋めるように伸びる。

コンテナの幅が足りなかったときの挙動flex-shrinkプロパティ

設定した比に即して要素が縮む

flex-basis,grow,shrinkの一括指定flexプロパティ

flex:1;をよく設定することがある。その場合grow:1,shrink,1,basis:0%で設定したことになる。

display:flex;とした時、その要素直下の子要素のみflex-itemとなる。※注意

footerを画面下に持っていく時に気を付ける事。

bodyタグ（コンテナ）内にheader,main,footerがありfooterを画面最下に持っていきたいとき。

mainのgrow:1としただけだとfooterが下に行かない。それはbodyのheightが定められていない為、

bodyの高さが要素の高さになっている、bodyのheightを100%にする必要が有る。

bodyの親要素であるhtml要素も100%にしなければならない。

flexboxのmarginについて

flexboxの縦方向のmarginは相殺されない点に注意

flexボックスを使って要素を中央に配置する。

コンテナにdisplay:flex; justify-content: center; align-items:center;と設定するだけ。

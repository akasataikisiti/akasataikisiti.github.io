複数のクラス属性

クラス属性は半角スペースで複数の付与ができる。id属性は一つしか付与できない。

属性セレクター２

属性の値が特定の物である要素を選択させるには[href="#top"]みたいに書いてやればよい。

クリックしたら#topに飛ぶ要素

また前方一致もつかうことができる。[href^="http"]の様に書いたらhref属性の値がhttpで始まる要素を選択できる。

後方一致は[href$="com"]の様に書いたらhref属性の値がcomで終る要素を選択できる。

部分一致は[href*="com"]の様に書いたらhref属性の値にcomが含まれる要素を選択できる。

セレクターの組み合わせ。

ある要素の直下のある要素

main > h1 でmain要素直下のh1を選択

あるセレクターの下の全ての階層のある要素

main h1 でmain要素下の全てのh1を選択

同じ階層である要素の次のある要素

p + p でp要素の次にあるp要素が選択される。

<p>Hello,hello.</p>
<p>Hello,hello.</p>
<p>Hello,hello.</p>
p + p {  border-top: 1px solid orange; }

カーソルを当てたときの状態にスタイルを設定する。

.btn:hoverでボタンクラスにホバーした状態にスタイル付けれる。

.btn:hover {curose:pointer;}としたらホバーしたときにマウスの形を変える事ができる。

:nth-child疑似クラス。

mainクラス直下に複数のpが並んでいる状態で３番目のpにスタイルを設定したい時。

main > n-th-child(3)としたらできる。

main > n-th-child(3n)とすることで３の倍数箇所に指定できる。

main > n-th-child(odd)とすることで奇数箇所の倍数箇所に指定できる。

main > n-th-child(even)とすることで偶数箇所の倍数箇所に指定できる。

main > first-child;とすることで最初の要素に指定できる。

main > last-child;とすることで最後の要素に指定できる。

:nth-of-type疑似クラス

ある要素の３番目の要素にスタイルを指定したいとき。

main > h2:nth-of-type(3)と指定できる。

main > first-of-type;とすることで最初の指定要素に設定できる。

main > last-of-type;とすることで最後の指定要素に設定できる。

empty

例えば、li:emptyと書いたら、li要素の中で空の物にスタイルを付ける事ができる。半角スペースは空にならないので注意

not

疑似クラスの否定を表す事ができる。

li:not(:empty)とすることで、空でないli要素を指定できる。

指定の詳細度

id class,属性,疑似クラス 要素,疑似要素の順に詳細度が高くなる。idが100の位、classとかが10の位、要素とかが1の位と考えたら分かり易い。

プロパティ値の設定がバッティングした場合、セレクターの詳細度が高い方の物が優先される。

ただしhtmlのstyle属性に直接書いたものがある場合そちらが優先される。

もっとも優先度が高い書き方は属性値の後ろに !importantをつけた値となる。

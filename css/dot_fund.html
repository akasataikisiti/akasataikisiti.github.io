cssの基本

記法

cssのプロパティと値をまとめて宣言という

style継承

cssのプロパティには子要素に継承されるものが有る、MDNで調べる事ができる。

また本来継承されないプロパティにもinherit値を設定することで継承させる事ができる。

＜例＞border:inherit;

fontの設定

cssファイル内に日本語が含まれる場合ファイルの先頭に@charset "utf-8;"を設定する

line-heightについて

pxで設定することもできるし、3emまたは3とすることでfont-sizeの３倍になる。

単位あり子要素に対して親要素で計算した数値がそのまま継承される。

単位無しの場合子要素でも数値が再計算した値が反映される。

vertical-align

vertical-alignのデフォルト値はbaselineに設定されている。

top,middle,bottom等の値がある。また-2pxとすることでbaselineから下に2pxと指定できる

透明度操作

例えばあるP要素全体の透明度を指定もopacity: 0.5;みたいに操作できる。

画像の透明度も操作できる。

ul内にlist-style-type: cirle;とする事で〇で箇条にできる

ol内にlist-style-type:lower-alpha;とすることで小文字英で箇条にできる。

またlist-style-type: none;とすることで箇条を消す事ができる。

list-style-image:url();で画像を箇条に使える。

リストの箇条の位置を指定することもできる。

list-style-position: inside;でできる

一括指定プロパティ

list-style: circle inside url();で一括指定できる

一括指定プロパティの前に個別で値を指定する場合は値が上書きされる可能性が有るので注意すること

ボックスの大きさ指定

width,heightでしていできる。pxでしていすることができる。

また%で指定することで、親要素に対する大きさの割合を指定することが出来る。

ボーダー指定

border-width:2px; border-style:solid; border-color:blue;を一括でしていすると

border: 2px solid blue;となる

border-topやborder-bottomなど特定の方向に付ける事もできる,またborderに指定した後にborder-bottom:noneとして下だけ消すこともできる。

またborder-radius:10px;とすることで、ボーダーを丸くすることもできる

marginの左右方向のauto

margin-left:auto;はボックスを指定した時にできる左右の余白を左に割り当てる意味になる。

margin-left:auto;,margin-right:autoとするとボックスが中央揃えになる。

marginの相殺

marginの垂直方向は互いに相殺される。ただし水平方向は相殺が起こらない。

displayプロパティ

全ての要素はdisplay:block,またはdisplay:inlineのものに分類される。

block要素はwidthが親要素いっぱい、高さがコンテンツの高さになる。inlineは両方ともコンテンツの数値になる。

displayの要素はwidth,heightが指定できない。したいときはdisplay:inline-block;としてあげる。

またdisplay:none;とすると要素を表示しないようにすることが出来る。

positionプロパティ

position: staticがデフォルト値。

position: relative;にしてtop:30px left:30pxとするとstaticの位置から下に30px,左に30pxずれた位置に配置される。

画面に要素を固定する。

position:fixed;としてtop:30px left:30pxとすると、ウィンドウの起点から下に30px,左に30pxの位置に配置され、スクロールしても固定されている。

またposition:fixedとした要素は他の要素の配置の影響から外れる。

position:absolue;

親要素がstaticの場合はウィンドウの起点からtop,left,right分ずれた位置に、配置

親要素がfixedやrelativeの場合はその親要素を起点にしてずれた位置に配置される。

要素の重なり順をしていする。z-indexプロパティ

z-indexが大きい値程上に表示される。

boxのwidth,heightをpadding,borderを含めた指定に変更

box-sizing:border-boxにするとできる。

width: calc((100% - 40px) / 3)で計算結果をwidthに自動設定

calc内の演算子の間にはスペースを入れる必要があるので注意

要素に影を付ける。

box-shadow: 10px 50x rgba(0,0,0,0.3)で右下方向に影を付けれる。

box-shadow: 10px 5px 3px rgba(0,0,0,0.3)と三つ目の数値で影をぼかすことが出来る。

box-shadow: 10px 5px 3px 10px rgba(0,0,0,0.3)と4つ目の数値で影を拡大出来る。

text-shadowにしてテキストに影を付けれる。

背景に画像を設定し、その画像の縦横比を保持したままできるだけbox内に大きく表示したいとき。

header {
  height: 80px;
  background-color: pink;
  background-image:url(../img/dot_fund_html.png);
  background-size: cover;
}

background-size:cover;で写真の縦横比をそのままにできる。

ここに関してはほんと使うたびに調べる必要が有る。レッスン32参照

floatプロパティ

画像とかを右に寄せる。 float:right;

floatを設定した要素はその後に配置される要素から無視される。

ただし。テキストが長くなりfloat要素に差し掛かった時は、テキストはその要素を回り込む。

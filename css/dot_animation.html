cssアニメーションの考え方。

変化前と変化後のスタイルを作って、その中間を補完するという手法

transformプロパティ

translateX(100px),translateY(100px)でそれぞれX,Yに移動。Translate(100px,100px)で両方移動

rotate(60deg)で回転。scaleX(2),scaleY(2)でそれぞれ拡大。scale(2,2)で両方拡大

transformは複数valueを設定できるが。順番が大事

先に設定した値から順に適応される。

<div class="box"></div>
.box {
  width: 100px; height: 100px; background:orange;
transition-property:transform;
transition-duration: 1s; 
}
.box {:hover
transform: translateX(100px) rotate(30deg);
}

rotateの起点

デフォルトだと図形中央を起点に回転するが、gransform-origin: top left;と書いたら左上を起点に回転する。

またtransform-originは変形後にだけ適用したいわけではないので、変形前のcssに記載する。

またscaleもでふぉるだと中央を起点に拡大するが、同じように起点変更できる。

色の変化

変化後のcssに変化後の色を追加してあげる。またtransition-propertyにbackgroundを追加してやる。

<div class="box"></div>
.box {
  width: 100px; height: 100px; background:orange;
  transition-property: transform, background;
  transition-duration: 1s; 
}
  .box {:hover
  transform: translateX(100px) rotate(30deg);
  background: skyblue;
}

transition-property:allとして全てのプロパティをアニメさせることもできる
